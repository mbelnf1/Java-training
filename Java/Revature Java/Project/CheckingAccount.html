Checking Account

  • getAccountID(): A method to get the account identification number to be able to locate the personal bank account
  • setAccountID(): A method to create or change a new account identification number for a bank account
  • getminBalance(): A method to get the minimum balance of an account to know if it will be acceptable for a withdrawal/deposit
  • getAccountType(): A method to choose the appropriate bank account for the transaction, checking or balance accounts
  • getFirstName(): A method to get the first name of the user
  • getLastName(): A method to get the last name of the use
  • getOverdraftFee(): A method to get the overdraftfee value from the bank
  • toString(): A method to update a string to show multiple functions and print it
